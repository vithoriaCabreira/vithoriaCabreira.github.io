# Projeto FrontEnd

## Descrição do Projeto

Este projeto consiste em uma aplicação web desenvolvida para fins educacionais, com foco em práticas de desenvolvimento Front-End. O objetivo é criar uma interface amigável, responsiva e funcional, utilizando HTML, CSS e JavaScript.

## Estrutura do Projeto

- **index.html**: Página principal de login.
- **home.html**: Página inicial após o login.
- **cadastro.html**: Página de cadastro de novos usuários.
- **css/**
  - **style.css**: Estilos principais das páginas de login e cadastro.
  - **principal.css**: Estilos gerais do site.
  - **cssCurriculumLeonardo.css**: Estilos específicos do currículo de Leonardo
  - **cssCurriculumLeticia.css**: Estilos específicos do currículo de Letícia
  - **cssCurriculumVithoria.css**: Estilos específicos do currículo de Vithoria
- **figs/**: Pasta para armazenar imagens utilizadas no projeto.
